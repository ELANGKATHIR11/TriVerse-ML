# Disease Prediction - App Screenshots & Walkthrough

This directory contains visual captures demonstrating the capabilities of the **Disease Prediction** engine within the **TriVerse AI** platform.

## Screenshot Directory

### `01_home.png`
- **Description**: The main dashboard screen showing global system health, CPU/GPU temperature, active environment parameters, and direct navigation keys to CreditVerse, HealthVerse, and VisionVerse modules.
- **Key Elements**: Glassmorphic widgets, system utilization graph, active model counts.

### `02_dataset.png`
- **Description**: The Dataset Inspector page showing stats for Heart Disease, Breast Cancer, and Pima Diabetes datasets.
- **Key Elements**: Outlier percentages, feature count, correlation heatmaps, feature histograms.

### `03_training.png`
- **Description**: Training monitor displaying real-time updates of the classifiers (Logistic Regression, SVM, Random Forest, XGBoost, CatBoost, MLP).
- **Key Elements**: Live terminal feedback, training loss charts, epoch completion rate.

### `04_results.png`
- **Description**: Model performance table listing and sorting all 6 classifiers based on their validation accuracy and ROC-AUC.
- **Key Elements**: Table containing model names, precision, recall, F1, training duration, and size of the saved pickle files.

### `05_comparison.png`
- **Description**: Comparative visualizations of the classifiers.
- **Key Elements**: Overlaid ROC curves, side-by-side confusion matrices, and parallel coordinate plots mapping hyperparameters.

### `06_prediction.png`
- **Description**: The clinical diagnosis interface. Users input variables (like cholesterol, blood pressure, fasting blood sugar, gender) to run predictions.
- **Key Elements**: Slider selectors, prognosis probability bar charts, and LIME local explanation blocks displaying risk factors.
