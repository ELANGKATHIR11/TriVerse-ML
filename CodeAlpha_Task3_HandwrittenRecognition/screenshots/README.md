# Handwritten Character Recognition - App Screenshots & Walkthrough

This directory contains visual captures demonstrating the capabilities of the **Handwritten Character Recognition** engine within the **TriVerse AI** platform.

## Screenshot Directory

### `01_home.png`
- **Description**: The main dashboard screen showing global system health, CPU/GPU temperature, active environment parameters, and direct navigation keys to CreditVerse, HealthVerse, and VisionVerse modules.
- **Key Elements**: Glassmorphic widgets, system utilization graph, active model counts.

### `02_dataset.png`
- **Description**: The MNIST/EMNIST Dataset explorer page showing random grids of digitized character images.
- **Key Elements**: Label selectors, pixel-density grids, image previews.

### `03_training.png`
- **Description**: Real-time training monitoring logs tracking PyTorch training loop metrics (Epoch loss and classification accuracy) on local CUDA computing.
- **Key Elements**: PyTorch log terminal, Epoch vs. Loss, Epoch vs. Accuracy line charts.

### `04_results.png`
- **Description**: Leaderboard comparing the custom CNN model and the ResNet18 model.
- **Key Elements**: Table containing model parameters, weight size in MBs, training time, and peak memory usage.

### `05_comparison.png`
- **Description**: Comparative analysis view for computer vision models.
- **Key Elements**: Heatmaps of the multi-class Confusion Matrix ($10\times 10$ for MNIST digits, $47\times 47$ for EMNIST characters), misclassifications chart.

### `06_prediction.png`
- **Description**: The interactive canvas drawing board where users paint characters and get real-time recognition results.
- **Key Elements**: Painting canvas, clear/predict buttons, classification confidence bar charts, and activation map overlays showing focus points.
