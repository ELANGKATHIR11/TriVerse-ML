# CodeAlpha Task 1 - Credit Scoring Model Submission

This folder contains the submission package for Task 1: Credit Scoring Model.

## Overview
A comprehensive credit default risk prediction system integrated into the TriVerse ML platform.

## Architecture
Integrates with the unified SQLite database, MLflow experiment dashboard, and Optuna HPO engine.

## Dataset
- Give Me Some Credit
- German Credit Data

## Models & Results
- Models: Logistic Regression, Decision Tree, Random Forest, CatBoost, MLP.
- Evaluated via F1-Score, Precision, Recall, and ROC-AUC.
- Logged natively to MLflow.
