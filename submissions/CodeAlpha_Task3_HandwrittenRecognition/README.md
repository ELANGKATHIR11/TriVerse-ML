# CodeAlpha Task 3 - Handwritten Character Recognition Submission

This folder contains the submission package for Task 3: Handwritten Character Recognition.

## Overview
A deep learning system for digit and alphabet recognition.

## Architecture
Integrates both TensorFlow-based custom CNNs and PyTorch-based ResNet18 models, managed by MLflow.

## Dataset
- MNIST Digits
- EMNIST Balanced

## Models & Results
- Custom CNN (Acc: ~98.5%)
- PyTorch ResNet18 (Acc: ~99.1%)
