# CodeAlpha Task 4 - Disease Prediction Submission

This folder contains the submission package for Task 4: Disease Prediction.

## Overview
Structured clinical prediction models for multiple medical classification tasks.

## Architecture
Integrates SHAP, Optuna tuning, and automated diagnostics with native PDF/DOCX/PPTX report export.

## Dataset
- UCI Heart Disease
- Pima Indians Diabetes
- Breast Cancer Wisconsin

## Models & Results
- SVM, XGBoost, Random Forest, CatBoost, MLP.
